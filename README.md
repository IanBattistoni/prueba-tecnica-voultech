# Prueba Tecnica API

API REST desarrollada en .NET 8 utilizando PostgreSQL.

## Tecnologías
- ASP.NET Core
- PostgreSQL
- Npgsql
- Git

## Endpoints

### Ordenes
- POST /ordenes
- GET /ordenes
- GET /ordenes/{id}
- PUT /ordenes/{id}
- DELETE /ordenes/{id}

### Productos
- POST /productos
- GET /productos

### Usuarios
- POST /users
- GET /users
- GET /users/{id}
